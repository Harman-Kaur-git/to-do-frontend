# ✅ Todo List App

A clean and minimal **Todo List** web app built with pure **HTML**, **CSS**, and **Vanilla JavaScript**. Tasks are saved in **localStorage** — no backend, no database, no login needed.

---

## 🌍 Live Demo

🚀 **[click here to get live demo](https://checktodofrontend.netlify.app)**

---

## 🚀 Features

- ➕ **Add Todos** — Quickly add new tasks
- 🗑️ **Delete Todos** — Remove tasks you no longer need
- 💾 **localStorage Persistence** — Tasks stay saved even after page refresh
- 📱 **Responsive Design** — Works on mobile, tablet, and desktop

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 |
| Logic | Vanilla JavaScript (ES6+) |
| Storage | localStorage |

> Zero dependencies. No npm. No build step. Just open and use.

---

## ▶️ Getting Started

No install needed.

```bash
git clone https://github.com/your-username/todo-list.git
cd todo-list
```

Open `index.html` in your browser — done.

---

## 🧠 How It Works

- Each todo gets a unique ID and is stored as an object in **localStorage**
- On page load, todos are read from localStorage and rendered to the DOM
- Deleting removes the item from localStorage and re-renders the list
